
# XYZ-UI-GamePlugin
A RocketMod plugin for Unturned servers that displays real-time player XYZ coordinates on the HUD/UI, helping players and admins track position information easily.

This plugin requires an additional UI component to function properly. Steam Workshop ID: 3730585666.
